# PDA 消息兜底提醒 APP — 现场部署检查清单

> 工业 PDA 上能否长期稳定提醒，**一半靠 APP，一半靠部署**。请按顺序逐项完成。

---

## 一、安装前确认

- [ ] PDA 系统版本为 **Android 7.1**
- [ ] 已确认 **Hi护士 实际包名**（设置 → 应用 → Hi护士 → 应用信息中查看，或咨询厂商）
- [ ] 本 APP 为 **Release APK** 安装（非调试版长期运行）

---

## 二、APP 内配置（安装后首次打开）

### 1. 权限引导（三项必做）

- [ ] **通知使用权**：设置 → 特殊权限 / 通知使用权 → 开启「PDA 消息兜底」
- [ ] **电池优化**：选择「不优化 / 无限制」
- [ ] **厂商自启动**：按 PDA 品牌完成下方对应步骤，并在 APP 内勾选「已完成」

### 2. 主界面设置

- [ ] **监控包名** 填写 Hi护士 实际包名（例如 `com.xxx.hinurse`，以现场为准）
- [ ] **提醒总开关** 打开
- [ ] **响铃 / 振动 / 锁屏亮屏** 按现场需求开启
- [ ] **通知停留** 建议选 **60 秒** 或 **手动清除**
- [ ] 点击 **测试按钮**，确认有声音、振动、兜底通知出现
- [ ] **保活状态** 显示：前台服务 ✅、通知监听 ✅

---

## 三、厂商 ROM 自启动与白名单（常见 PDA）

> 不同品牌菜单名称略有差异，请在「设置」或「安全中心」中查找类似项。

| 检查项 | 操作 |
|--------|------|
| 自启动管理 | 允许「PDA 消息兜底」自启动 |
| 后台运行 / 后台活动 | 允许后台运行，不限制 |
| 电池 / 省电 | 不优化、不受限、允许后台高耗电 |
| 最近任务 | 打开 APP 后，在最近任务中 **锁定**（下拉或长按锁定图标） |
| 受保护应用 / 纯净后台 | 将本 APP 加入受保护列表 |

### 优博讯 / 新大陆 / 霍尼韦尔等工业 PDA

- [ ] 在厂商「MDM / 设备管理」中确认未禁止 Notification Listener
- [ ] 确认未禁止第三方 APP 后台服务
- [ ] 如有「企业省电策略」，将本 APP 加入例外

---

## 四、Hi护士 侧建议（需与业务方协调）

本 APP **无法替代** Hi护士 自身收消息。若仍频繁漏消息，请同时排查：

- [ ] Hi护士 是否同样加入电池白名单与自启动
- [ ] Hi护士 在漫游后是否能恢复连接（WLAN 问题，非本 APP 范围）
- [ ] Hi护士 被杀后是否还能把通知投递到系统栏（若不能，需厂商改 Hi护士）

---

## 五、验收测试（现场 15 分钟）

1. **即时测试**：在 APP 内点「测试兜底通知」→ 应有声、振、亮屏、兜底通知
2. **联调测试**：由同事向 Hi护士 发一条真实消息 → 应出现 **双通知**（Hi护士 原通知 + 兜底通知）
3. **锁屏测试**：锁屏静置 30 分钟 → 再发 Hi护士 消息 → 仍应触发兜底
4. **负向确认**：强制停止 Hi护士 且无任何系统通知 → 本 APP 不应误报（也 **不会** 凭空提醒）

---

## 六、日常运维

| 现象 | 排查 |
|------|------|
| 完全无兜底 | 检查通知使用权是否被系统关闭；重开权限 |
| 有声无振 / 有振无声 | 检查 PDA 音量、勿扰模式；APP 内开关 |
| 锁屏不亮 | 开启「锁屏亮屏」；部分 ROM 需额外允许「锁屏显示」 |
| 用几天后又失效 | 最近任务是否被划掉；省电策略是否重置；重新锁定 + 白名单 |
| Hi护士 有消息但从未进系统栏 | **非本 APP 问题**，查 Hi护士 与网络 |

---

## 七、卸载 / 重装

重装后需 **重新完成第二节全部配置**。建议保存本文档 PDF 供现场人员使用。
